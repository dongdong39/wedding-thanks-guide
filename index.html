GitHub 회원가입

GitHub은 무료 웹페이지를 만들 수 있는 서비스입니다.

  1. github.com에 접속합니다.
  2. Sign up 버튼을 눌러 회원가입합니다.
  3. 이메일, 비밀번호, 사용자 이름을 입력합니다.
  4. 이메일 인증을 완료합니다.
💡 이미 GitHub 계정이 있다면 이 단계를 건너뛰세요!

감사카드 파일 다운로드

감사카드 템플릿 파일을 다운로드합니다.

  1. 아래 링크를 클릭하여 템플릿을 다운로드합니다.
📥 감사카드 템플릿 다운로드
💡 녹색 Code 버튼 → Download ZIP을 클릭하면 됩니다.

파일 구성

📁 wedding-thanks-template/
├── index.html ← 신랑신부용 감사카드

    ├── hero.jpg ← 감사 인사 이미지
    ├── g1.jpg ~ g16.jpg ← 갤러리 사진들

감사 인사말 수정하기

index.html 파일을 메모장이나 텍스트 편집기로 열어 수정합니다.

바꿔야 할 부분들

1) 상단 커플 이름과 날짜

<!-- 이 부분을 찾아서 수정하세요 -->

<p class="hero-date">2026. 05. 09</p>

2) 인사말 본문 — 원하는 내용으로 자유롭게 수정하세요.

<div class="greeting-text">
  <p>저희 두 사람의 새로운 시작을<br>따뜻한 마음으로 축복해주신<br>모든 분들께 깊이 감사드립니다.</p>
  <!-- 줄바꿈은 <br>을 사용하세요 -->
  <!-- 문단 구분은 <div class="greeting-line"></div>를 사이에 넣으세요 -->
</div>

3) 서명 (이름, 혼주)

<div class="greeting-sign">
  <p>하망이 · 하망이</p>
  <p class="greeting-parents">신랑측 아버지 · 어머니</p>
  <p class="greeting-parents">신부측 아버지 · 어머니</p>
  <p class="greeting-olim">올림</p>
</div>

4) 페이지 제목 (카카오톡 공유 시 표시)

<title>하망이🤍하망이 감사카드</title>
<meta property="og:title" content="하망이🤍하망이 감사카드">
✅ 수정할 때 <p>, </p> 같은 태그는 건드리지 말고 글자만 바꿔주세요!

GitHub에 올리기

  1. github.com/new에서 새 레포지토리를 만듭니다.
  2. Repository name에 wedding-thanks를 입력합니다.
  3. Public을 선택합니다. (무료 배포를 위해 필수!)
  4. Create repository 버튼을 클릭합니다.
  5. 생성된 페이지에서 "uploading an existing file" 링크를 클릭합니다.
  6. 다운받은 파일들을 모두 드래그 앤 드롭합니다.
⚠️ images 폴더째로 드래그하세요! 파일을 하나씩 넣으면 폴더 구조가 깨집니다.
💡 업로드 후 하단의 Commit changes 버튼을 꼭 클릭하세요!
7

웹페이지 배포하기 (GitHub Pages)

  1. 레포지토리 페이지에서 상단 Settings 탭을 클릭합니다.
  2. 왼쪽 메뉴에서 Pages를 클릭합니다.
  3. Source 항목에서 Deploy from a branch를 선택합니다.
  4. Branch를 main으로 선택하고 Save를 클릭합니다.
  5. 1~2분 후 상단에 초록색 링크가 나타납니다!
✅ 링크 형식: https://사용자이름.github.io/wedding-thanks/

자주 묻는 질문

index.html에서 갤러리 부분을 찾아 <div class="gallery-item">...</div> 줄을 추가하거나 삭제하세요. 4의 배수로 맞추면 깔끔합니다. 파일 이름은 g17.jpg, g18.jpg 처럼 순번을 이어가면 됩니다.

index.html에서 <details class="ceremony-item"> 블록을 삭제하거나, <summary> 안의 텍스트와 <source src="...">의 파일 이름을 수정하세요.

GitHub Pages 반영에 1~2분 정도 걸립니다. 또한 브라우저 캐시 때문에 안 보일 수 있으니, 시크릿 모드나 강력 새로고침(Ctrl+Shift+R)을 시도해보세요.

온라인 무료 영상 압축 사이트(예: FreeConvert)를 사용하거나, 핸드폰에서 영상을 다시 내보낼 때 해상도를 낮추면 됩니다. 5MB 이하를 추천합니다.

네! GitHub Pages는 완전 무료입니다. 도메인도 사용자이름.github.io 형태로 무료 제공됩니다. 기간 제한도 없습니다.

GitHub에서 해당 레포지토리를 삭제하면 페이지도 함께 사라집니다. Settings → Danger Zone → Delete this repository
